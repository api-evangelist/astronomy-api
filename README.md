# AstronomyAPI (astronomy-api)
AstronomyAPI is a web API for retrieving astronomical information including data about celestial bodies, moon phases, planet positions, star charts, and astronomical events for a given location and time.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/astronomy-api/refs/heads/main/apis.yml)
